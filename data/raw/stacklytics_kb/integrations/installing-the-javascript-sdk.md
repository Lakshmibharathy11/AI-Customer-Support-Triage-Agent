# Installing the JavaScript SDK

## Introduction to the JavaScript SDK
The JavaScript SDK is one of the available SDKs for Stacklytics, along with Python and Go. It allows engineering teams to easily integrate Stacklytics into their applications and start tracking events. To get started, you can find the JavaScript SDK on our website, and it's included in all our customer tiers, including Gold, Diamond, and Platinum. The JavaScript SDK is a powerful tool that enables you to track events and gain valuable insights into your application's performance.

## Installing the JavaScript SDK
To install the JavaScript SDK, you'll need to include it in your project and initialize it with your API key. You can find your API key in the Stacklytics dashboard, and it's essential to keep it secure to avoid any unauthorized access. If you encounter any issues during installation, you may see error codes such as 401, which indicates an invalid API key. Make sure to check the API base URL, https://api.stacklytics.io/v2, to ensure you're using the correct endpoint.

## Troubleshooting and Support
If you encounter any issues while installing or using the JavaScript SDK, you can refer to our documentation or reach out to our support team. We're available to assist you Monday through Friday, 9am-5pm Pacific Time, and our response times vary depending on your tier, with Platinum customers receiving a response within 1 hour. Common errors such as SDK-1042, which indicates an event schema validation failure, can be resolved by checking your event schema and ensuring it aligns with our requirements. You can also check our status page, https://status.stacklytics.io, for any updates on our system's performance and uptime.
