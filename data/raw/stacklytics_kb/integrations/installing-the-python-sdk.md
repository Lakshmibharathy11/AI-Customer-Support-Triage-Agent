# Installing the Python SDK

## Introduction to the Python SDK
The Stacklytics Python SDK is one of the available SDKs, along with JavaScript and Go, designed to help engineering teams integrate our product-analytics SaaS into their applications. To get started, you can find the SDK on our website or through your preferred package manager. The Python SDK is compatible with our API base URL, https://api.stacklytics.io/v2, and is available for all customer tiers, including Gold, Diamond, and Platinum. Before installing, ensure you have a valid API key to avoid common errors like 401, which indicates an invalid API key.

## Installation and Setup
To install the Python SDK, you can use pip, the Python package manager, by running a simple command in your terminal. Once installed, you'll need to initialize the SDK with your API key, which can be found in your Stacklytics account settings. If you encounter any issues during setup, such as event schema validation failures (SDK-1042), refer to our documentation or contact support for assistance. Support is available Monday through Friday, 9am-5pm Pacific Time, with response times varying by tier, from 8 hours for Gold to 1 hour for Platinum.

## Troubleshooting and Resources
If you encounter any errors during installation or usage, such as rate limit exceeded (429, error code RATE_LIMIT_EXCEEDED), check our status page at https://status.stacklytics.io for any ongoing issues. For more detailed information on the Python SDK, including code examples and troubleshooting guides, visit our documentation portal. Remember, as a Stacklytics customer, you're also entitled to support based on your tier, whether it's standard support for Gold or dedicated support for Platinum, with priority and response times adjusted accordingly.
